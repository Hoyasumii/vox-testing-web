## 🚀 **Como Usar**

### **Instalação:**
```bash
pnpm install
```

### **Executar em desenvolvimento:**
```bash
pnpm dev
```

### **Executar testes:**
```bash
pnpm test:run
```

### **Build para produção:**
```bash
pnpm build
```

### **Verificar código:**
```bash
pnpm typecheck  # Verificar TypeScript
pnpm lint       # Verificar linting
pnpm format     # Formatar código
```
